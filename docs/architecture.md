# 系统架构

## 概述

金融多智能体协作系统基于 LangGraph StateGraph 构建，包含四个专职 Agent 节点，通过条件边实现动态路由与自我反思。

## 架构图


## 核心组件

### Agent 节点

| Agent | 职责 | 工具 |
|-------|------|------|
| 数据收集 | Wind API 抓取、PDF 解析 | Wind 终端、PyMuPDF |
| 财务分析 | 杜邦分解、现金流异动 | 自定义计算引擎 |
| 舆情解读 | 新闻/论坛情感分析 | LLM + 提示词 |
| 校验总结 | 事实核对、报告生成 | DeepSeek-V3 |

### State 设计

```python
class AgentState(TypedDict):
    task_id: str
    company_code: str
    report_date: str
    raw_data: dict           # 数据收集结果
    financial_metrics: dict   # 财务分析结果
    sentiment_result: dict    # 舆情分析结果
    draft_report: str         # 报告草稿
    errors: list[str]         # 错误收集
    retry_count: int          # 重试计数（≤3）
    status: str               # pending/running/done/failed
```

### 条件边规则

- `retry_count < 3 AND errors 非空` → 路由至重写节点
- `retry_count >= 3 OR errors 为空` → 路由至输出节点
